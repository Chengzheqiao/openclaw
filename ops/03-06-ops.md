## Why This Took Longer Than Expected

- The upstream delta was very large (fork far behind upstream).
- Conflict areas overlapped with local customizations.
- Environment drift added noise:
  - default Node was `v25`
  - repo stable line is `Node 22`
  - UI build behavior was unstable under the active environment
- Codex command sessions do not preserve background child processes reliably.
- Browser auth failures looked like transport disconnects until logs were checked.

## Key Lessons

- Pin runtime first, then install/build/test.
- Separate merge conflicts from environment failures as early as possible.
- Do not open the Control UI from a bare loopback URL when token auth is enabled.
- For this repo on this machine, `nvm use` should be the first step in local dev.
